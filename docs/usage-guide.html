01 — ログイン

管理画面へのログイン

メールアドレスまたはGoogleアカウントで認証します。

  • A
    メール＆パスワード — メールアドレスとパスワードを入力して「ログイン」。パスワードは表示/非表示を切り替えられます。
  • B
    Googleアカウント — 「Googleでログイン」ボタンから既存のGoogleアカウントで簡単にログインできます。
TIP
パスワードを忘れた場合：ログイン画面下部の「パスワードを忘れた方」から再設定メールを送信できます。 新規登録は「アカウント作成」リンクから。メール認証後にログイン可能になります。

03 — ダッシュボード

ダッシュボード

ログイン後に最初に表示される画面。売上・注文・在庫をリアルタイムで把握できます。

売上

期間内の合計売上額。前期比トレンドも表示。

注文数

受注件数とステータス別内訳。未対応の注文がひと目でわかります。

商品数

登録商品の総数と在庫状況（正常 / 残りわずか / 在庫切れ）。

顧客数

総顧客数と新規顧客の推移。リピート率も確認できます。

月別売上推移グラフは「月次」「年次」「全期間」から切り替え可能。最近の注文・在庫アラート・人気商品もウィジェット形式で表示されます。 各ウィジェットはドラッグ＆ドロップで並び替えられます。

04 — 商品管理

商品管理

商品の登録・編集・在庫管理・カテゴリー管理をひとつの画面で行います。

4-1. 商品一覧

  • 検索
    商品名またはSKUで絞り込み。リアルタイムでフィルタリングされます。
  • 絞込
    ステータス（公開中 / 下書き / アーカイブ）やカテゴリーでフィルター。
  • 追加
    右上「+ 商品を追加」ボタンで新規登録画面へ。
  • CSV
    CSVファイルから商品を一括インポート。大量商品の登録に便利。

商品カード：緑 = 公開中、グレー = 下書き。在庫切れは赤、残りわずかはスカイブルーで表示。

4-2. 商品の登録・編集

左側のフォームと右側のリアルタイムプレビューで分割表示されます。

  • 1
    基本情報 — 商品名（必須）、短い説明、詳細説明を入力。
  • 2
    メディア — 商品画像をドラッグ＆ドロップでアップロード。PNG / JPG / WEBP / GIF（最大10MB）。
  • 3
    バリエーション — サイズ・カラーなどを設定。各バリエーションにSKU・価格・在庫数を設定できます。
  • 4
    カスタムフィールド — 組織で設定した独自入力項目（テキスト、数値、リストなど）。
  • 5
    SEO設定 — SEOタイトルとメタディスクリプションで検索対策。
TIP
「下書き保存」で途中保存、「公開する」でショップに即時反映されます。

4-3. 在庫管理

上部タブ「在庫」に切り替えると全商品の在庫状況を一覧で確認できます。「入出庫」タブでは在庫の増減履歴を時系列で確認。複数商品の在庫一括更新にも対応しています。

4-4. カテゴリー管理

「カテゴリー」タブからカテゴリーの作成・編集・削除が行えます。ショップのナビゲーションと商品フィルタリングに使用されます。

07 — コンテンツ管理

コンテンツ管理

ニュース・ブログ・特集・FAQなどの記事をリッチエディタで作成・公開します。

ニュース

お知らせ、プレスリリース、更新情報など。

特集

商品の特集記事、ピックアップ情報。

ブログ

ブログ記事、コラム、読み物。

FAQ

よくある質問とその回答。

コンテンツの作成手順

  • 1
    「+ コンテンツを作成」からタイプ（ニュース / 特集 / ブログ等）を選択。
  • 2
    リッチテキストエディタ（Tiptap）でタイトルと本文を入力。文字色・ハイライト・画像挿入・リンク設定が可能。
  • 3
    アイキャッチ画像を設定。一覧表示やSNSシェア時に使用されます。
  • 4
    公開設定を選択 — 下書き / レビュー依頼 / 公開 / アーカイブ。予約公開にも対応。

08 — 顧客管理

顧客管理

顧客情報の管理・分析・コミュニケーションを一元化します。

総顧客数

登録されている全顧客の数。

今月の新規

今月に新規登録された顧客数。

リピート率

リピート購入している顧客の割合。

平均購入額

顧客1人あたりの平均購入金額。

顧客タイプ（タブフィルター）

  • 全て
    全顧客を表示。
  • 個人
    個人顧客のみ表示。
  • B
    バイヤー — B2B取引先（購入者）を表示。
  • S
    サプライヤー — 代理店・卸売先を表示。

主な操作

  • 追加
    氏名・メール・電話番号・会社名・住所などの基本情報を入力して登録。
  • 検索
    名前・メール・会社名で検索。日付範囲フィルターも利用可能。
  • 詳細
    顧客名をクリックするとプロフィール・購入履歴・連絡先情報が表示。
  • 連絡
    顧客に直接メッセージを送信できます。

09 — マーケティング

マーケティング

アナリティクス・メルマガ・メッセージ・イベント・問い合わせ管理の5機能を搭載。

アナリティクス

商品ごとの閲覧数・クリック数・CTRを分析。月別グラフで推移を確認し、人気商品ランキングも表示。商品フィルターで特定商品に絞った分析が可能です。

メルマガ

件名と本文を入力し、送信先（全員 / 特定サプライヤーの顧客）を選択して一括送信。送信履歴も確認できます。

注意
メルマガ送信前に「設定 → メールドメイン」でドメイン認証を完了してください。

メッセージ

サプライヤーを選択してバイヤーに直接メッセージを送信できます。

イベント

展示会・セミナーなどのイベント情報を管理。タイトル・日時・場所・説明・ジャンル・地域タグを設定して公開します。

問い合わせ

顧客からの問い合わせをスレッド形式で管理。ステータス（オープン / クローズ）でフィルタリングし、チャット形式でやり取りを続けられます。

10 — 代理店管理

代理店管理

代理店パートナーの登録・コード発行・売上管理を一元化します。

  • 一覧
    登録済みの代理店を一覧で確認。ステータス（アクティブ / 停止中 / 審査中）でフィルタリングできます。
  • 登録
    「+ 代理店を登録」から新しい代理店を追加。カスタムフィールドで追加情報も管理可能。
  • 実績
    代理店ごとの売上・コミッション情報を追跡。レポートを自動生成します。

11 — 設定

設定

カテゴリー別のカードグリッドで構成。各カードをクリックすると詳細設定ページに遷移します。

ストア設定

  • 組織
    組織設定 — 会社名・住所・ロゴ・テーマカラーなどの基本情報。
  • 機能
    機能設定 — 通知ボックス・紹介コード・コンテンツタイプなどのON/OFF。

チーム＆アカウント

  • MB
    メンバー管理 — チームメンバーの招待・権限設定。ロールベースのアクセス制御。
  • AC
    アカウント設定 — パスワード変更・通知設定・セキュリティ。
  • RL
    ロール管理 — カスタムロールの作成と権限の詳細設定。

メール＆通知

  • DM
    メールドメイン — 送信元ドメインの認証設定（DNS設定が必要）。
  • TM
    メールテンプレート — 注文確認・発送通知などの自動メールをカスタマイズ。

連携＆決済

  • API
    API / Webhook — APIキーの発行・管理。外部サービスとの連携用Webhook URLを設定。
  • PAY
    決済設定 — Stripe連携の設定。クレジットカード・銀行振込・代金引換の有効/無効を切り替え。
  • PL
    プラン・請求 — 現在のプラン確認・変更・請求履歴。

ショップ外観

  • TH
    ショップテーマ — ヒーロー画像・ブランドカラー・セクション表示設定。
  • NV
    ナビゲーション — ショップのメニュー項目と表示順序をカスタマイズ。
注意
APIキーは公開リポジトリやフロントエンドのコードに含めないでください。サーバーサイドからのみ使用してください。

12 — ショップ

ショップ（顧客向けストアフロント）

顧客が商品を閲覧・購入するための公開ページ。管理画面の設定と連動します。

12-1. ショップトップページ

「設定 → ショップテーマ」でセクションの表示/非表示や順序を変更できます。

  • HV
    ヒーローバナー — 大きな画像でブランドの世界観を表現。
  • PR
    注目製品 — ピックアップした商品を4列グリッドで表示。
  • CT
    カテゴリー — 商品カテゴリーをビジュアルカードで表示。クリックで絞り込み。
  • B2B
    法人向けセクション — 法人会員登録や見積依頼のCTA。
  • NW
    記事・ニュース — 最新の記事やお知らせを表示。

12-2. 商品一覧ページ

全商品をグリッド表示（モバイル2列 / タブレット3列 / デスクトップ4列）。カテゴリータブで絞り込み、並び替え（人気順 / 新着順 / 価格順）が可能。 「FEATURED」「SALE」「SOLD OUT」バッジで商品状態がひと目でわかります。

12-3. 商品詳細ページ

  • 画像
    メイン画像＋サムネイル。矢印キーまたはサムネイルクリックで切り替え。
  • BV
    サイズ・カラーなどをボタンで選択。在庫切れはグレーアウト。
  • 数量
    +/- ボタンで調整。在庫5個以下の場合「残りX点」を表示。
  • カート
    ボタンクリックでカートに追加。トースト通知で確認。

12-4. カート・チェックアウト

  • 送料
    5,000円以上で送料無料。進捗バーで「あと ¥X,XXX で送料無料！」を表示。
  • CP
    クーポンコードで割引適用。（デモ: WELCOME10 で10%オフ）
  • 住所
    郵便番号7桁入力で都道府県・市区町村が自動補完。
クレジットカード

Visa / Mastercard / JCB / AMEX 対応。Stripeセキュア決済。

銀行振込

振込先情報を案内。入金確認後に発送。

代金引換

手数料 ¥330。商品受取時にお支払い。
