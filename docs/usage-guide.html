1. ログイン

管理画面にアクセスするための認証方法

📸
ログイン画面のスクリーンショット
screenshots/login.png を配置してください

ログイン方法は2つ

  • A
    メール＆パスワードでログイン
    メールアドレスとパスワードを入力して「ログイン」ボタンをクリック。パスワードは目のアイコンで表示/非表示を切り替えられます。
  • B
    Googleアカウントでログイン
    「Googleでログイン」ボタンをクリックすると、Googleの認証画面に遷移します。既存のGoogleアカウントで簡単にログインできます。
💡
パスワードを忘れた場合

ログイン画面下部の「パスワードを忘れた方」リンクからパスワードリセットメールを送信できます。

ℹ️
初回アカウント作成

新規登録は「アカウント作成」リンクから行えます。メール認証後にログインが可能になります。

3. ダッシュボード

売上・注文・在庫を一目で把握

📸
ダッシュボード画面のスクリーンショット
screenshots/dashboard.png を配置してください

ログイン後に最初に表示される画面です。ストアの状況をリアルタイムで確認できます。

KPIカード（上部）

💰 売上

期間内の合計売上額。前期比のトレンド（上昇↑/下降↓）も表示。

📦 注文数

受注件数とステータス別の内訳。未対応の注文がひと目でわかります。

🏷️ 商品数

登録商品の総数と在庫状況（正常/残りわずか/在庫切れ）。

👥 顧客数

総顧客数と新規顧客の推移。リピート率も確認できます。

売上チャート

月別の売上推移をグラフで表示。売上額と注文件数の二軸グラフで、トレンドを視覚的に把握できます。 期間は「月次」「年次」「全期間」から選択可能です。

最近の注文・在庫アラート

  • 📋
    最近の注文 — 直近の注文がステータスバッジ付きで一覧表示。クリックで注文詳細へ。
  • ⚠️
    在庫アラート — 在庫が少なくなった商品を警告表示。すぐに補充対応ができます。
  • 🏆
    人気商品 — 閲覧数・クリック数でランキングされた人気商品一覧。
💡
ウィジェットはドラッグで並び替え可能

ダッシュボードのウィジェットはドラッグ＆ドロップで自由に配置を変更できます。自分に合ったレイアウトにカスタマイズしましょう。

4. 商品管理

商品の登録・編集・在庫管理・カテゴリー管理

📸
商品一覧画面のスクリーンショット
screenshots/products-list.png を配置してください

4-1. 商品一覧

登録済みの全商品をグリッド表示で確認できます。

  • 🔍
    検索 — 商品名またはSKUで商品を検索。リアルタイムで結果がフィルタリングされます。
  • 🎛️
    フィルター — ステータス（公開中/下書き/アーカイブ）やカテゴリーで絞り込み。フィルターボタンで展開/折りたたみ。
  • ➕
    商品を追加 — 右上の「+ 商品を追加」ボタンで新規商品登録画面へ。
  • 📥
    CSV一括登録 — CSVファイルから商品を一括インポート。大量商品の登録に便利。

商品カードの見方

  • ●
    緑の丸 = 公開中 / グレーの丸 = 下書き / 薄いグレー = アーカイブ
  • !
    赤い在庫表示 = 在庫切れ / スカイブルー = 残りわずか / グレー = 在庫十分
📸
商品登録画面のスクリーンショット
screenshots/products-new.png を配置してください

4-2. 商品の登録・編集

新規商品の登録は、左側のフォームと右側のリアルタイムプレビューの分割画面で行います。

入力項目

  • 1
    基本情報 — 商品名（必須）、短い説明、詳細説明を入力。
  • 2
    メディア — 商品画像をドラッグ＆ドロップまたはクリックでアップロード。PNG/JPG/WEBP/GIF対応（最大10MB）。
  • 3
    バリエーション — サイズ・カラーなどのバリエーションを設定。「手動で追加」と「スウォッチ」の2つのモードがあります。各バリエーションにSKU、価格、在庫数を設定できます。
  • 4
    カスタムフィールド — 組織で設定した独自の入力項目（テキスト、数値、リストなど）。
  • 5
    SEO設定 — SEOタイトルとメタディスクリプションでSEO対策。

右サイドバー

  • 👁️
    リアルタイムプレビュー — 入力内容がショップでどう表示されるかをリアルタイムで確認。モバイル/デスクトップ表示を切り替え可能。
  • 📋
    公開設定 — ステータスを「下書き/公開中/アーカイブ」から選択。
  • 📁
    カテゴリー — チェックボックスでカテゴリーを選択。複数選択可。
  • 🏷️
    タグ — 自由にタグを追加・削除。商品の分類に活用できます。
💡
下書き保存と公開

「下書き保存」で途中保存、「公開する」でショップに即座に反映されます。

4-3. 在庫管理

商品一覧の上部タブから「在庫」タブに切り替えると、在庫専用ビューが表示されます。

  • 📊
    在庫一覧 — 全商品の在庫状況を一覧で確認。在庫切れ・残りわずかの商品がハイライト表示。
  • 📈
    入出庫履歴 — 「入出庫」タブで在庫の増減履歴を時系列で確認。誰がいつ変更したかも記録されます。
  • 📦
    一括操作 — 複数商品の在庫を一括で更新。大量の入荷処理に便利。

4-4. カテゴリー管理

「カテゴリー」タブから商品カテゴリーの作成・編集・削除が行えます。カテゴリーはショップのナビゲーションと商品フィルタリングに使用されます。

9. マーケティング

分析・メルマガ・メッセージ・イベント・問い合わせ管理

📸
マーケティング画面のスクリーンショット
screenshots/marketing.png を配置してください

マーケティングページは5つのタブに分かれています。

タブ 1: アナリティクス

商品ごとの閲覧数・クリック数・CTR（クリック率）を分析。月別のグラフで推移を確認し、人気商品ランキングも表示されます。 商品フィルターで特定の商品に絞った分析も可能です。

タブ 2: メルマガ

顧客にメールマガジンを一括送信。件名と本文を入力し、送信先（全員/特定サプライヤーの顧客）を選択して送信します。 送信履歴も確認できます。

⚠️
メールドメインの認証が必要

メルマガ送信前に、設定 → メールドメインで送信元ドメインの認証を完了してください。

タブ 3: メッセージ

個別またはグループへのメッセージ送信機能。サプライヤーを選択して、バイヤーに直接メッセージを送れます。

タブ 4: イベント

展示会やセミナーなどのイベント情報を管理。タイトル、日時、場所、説明、ジャンル、地域タグを設定して公開します。 バイヤーへの通知機能も搭載。

タブ 5: 問い合わせ

顧客からの問い合わせをスレッド形式で管理。ステータス（オープン/クローズ）でフィルタリングし、 チャットのようなUIでやり取りを続けられます。既読ステータスやファイル添付にも対応。

12. ショップ（顧客向けストアフロント）

ショップトップページのスクリーンショット

12-1. ショップトップページ

顧客が最初にアクセスするページです。管理画面の「ショップテーマ」設定でセクションの表示/非表示や順序を変更できます。

  • 🖼️
    ヒーローバナー — 大きな画像でブランドの世界観を伝えます。
  • ⭐
    注目製品 — ピックアップした商品を4列グリッドで表示。
  • 📂
    カテゴリー — 商品カテゴリーをビジュアルカードで表示。クリックでフィルタリング。
  • 🔥

  • 🏢
    法人向けセクション — B2B顧客向けの案内。法人会員登録や見積依頼のCTA。
  • 📰
    記事・ニュース — 最新の記事やお知らせを表示。

12-2. 商品一覧ページ

全商品をグリッド表示（モバイル2列、タブレット3列、デスクトップ4列）。カテゴリータブで絞り込み、並び替え（人気順/新着順/価格順）が可能。 「FEATURED」「SALE」「SOLD OUT」のバッジで商品状態がひと目でわかります。

12-3. 商品詳細ページ

  • 🎨
    バリエーション選択 — サイズ・カラーなどをボタンで選択。在庫切れのバリエーションはグレーアウト。
  • 🔢
    数量選択 — +/- ボタンで数量を調整。在庫が5個以下の場合「残りX点」の警告表示。
  • 🛒
    カートに追加 — ボタンクリックでカートに商品を追加。トースト通知で確認。
  • ❤️
    お気に入り — ハートアイコンでお気に入りに追加/削除。
  • 📤
    シェア — Web Share APIまたはURLコピーで商品を共有。

12-4. ショッピングカート

カートに追加した商品を確認・数量変更・削除できます。

  • 📋
    カート内容 — 商品画像、名前、バリエーション情報、SKU、価格が表示。数量の変更や個別削除が可能。
  • 🚚
    送料無料プログレスバー — 5,000円以上で送料無料。現在の進捗をバーで表示。「あと ¥X,XXX で送料無料！」
  • 🏷️
    クーポン — クーポンコードを入力して割引を適用。（デモ: WELCOME10 で10%オフ）
  • 💰
    注文サマリー — 小計、送料、合計を表示。「レジに進む」ボタンでチェックアウトへ。

12-5. チェックアウト（注文手続き）

  • 📮

  • 🏠

振込先情報が案内されます。入金確認後に発送。

手数料 ¥330。商品受取時にお支払い。
