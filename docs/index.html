akinai とは

EC機能をAPIで提供するサービスです

自社サイトやアプリから akinai のAPIを呼び出すだけで、本格的なEC機能が使えます。

商品管理

商品・在庫・カテゴリ

管理画面から商品を登録・編集し、APIで一覧や詳細データを取得できます。

購入フロー

カート・チェックアウト

在庫確認・金額計算・Stripe決済まで、購入の流れ全体をAPIで処理します。

コンテンツ

ニュース・ページ配信

ニュースやブログ記事などのコンテンツをAPIで取得して表示できます。

顧客・注文

注文管理・顧客情報

注文の状態確認や顧客情報の取得など、運営に必要な機能をAPI経由で利用できます。

API一覧

利用できるAPI

akinai が提供するすべてのAPIです。

メソッド エンドポイント できること
GET /api/v1/products 商品一覧の取得（検索・絞り込み対応）
GET /api/v1/products/:id 商品詳細の取得
GET /api/v1/categories カテゴリ一覧の取得
GET /api/v1/contents ニュース・ブログ等のコンテンツ一覧
GET /api/v1/contents/:id コンテンツ詳細の取得
POST /api/v1/cart/validate カートの在庫確認・金額計算
GET /api/v1/shipping-options 送料オプションの取得
POST /api/v1/checkout Stripe決済セッションの作成
POST /api/v1/orders 注文の作成
GET /api/v1/customers 顧客一覧・検索
